Lession Overview
    1. 3 Ways to include JavaScript
        1. Inline 
        Example:
        <head>
            <title>Inline JavaScript</title>
            <script>
                function myFunction() {
                    alert('Hello, World!');
                }
            </script>
        </head>
        2. Internal
        Example:
        <head>
            <title>Internal JavaScript</title>
            <script src="script.js"></script>
        </head>
        <body>
            <button onclick="myFunction()">Click me</button>
        </body>

        3. External
        Example:
        <head>
            <title>External JavaScript</title>
            <script src="script.js"></script>
        </head>
        <body>
            <button onclick="myFunction()">Click me</button>
        </body>
